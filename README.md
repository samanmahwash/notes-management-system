# Notes Management System

## Tech Stack
- React
- Tailwind CSS
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features
- User Authentication
- Create, Edit & Delete Notes
- Rich Text Editor
- Categories
- Search & Filter
- Pin & Archive Notes
- Responsive Dashboard
- Dark Mode

## Installation

### Backend

cd server
npm install
npm start

### Frontend

cd client
npm install
npm run dev

## Environment Variables

Create a `.env` file in the `server` folder and add:

PORT=
MONGO_URI=
JWT_SECRET=
CLIENT_URL=
